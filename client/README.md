# Blood Bank System - Client

A modern React application for a blood donation platform built with Vite.

## Features

- User registration and authentication
- Browse blood donors by group and location
- Send blood donation requests
- Accept/reject incoming requests
- Responsive design with animations

## Tech Stack

- React 19
- Vite
- React Router
- Axios
- Framer Motion
- Tailwind CSS

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Environment

Make sure the backend server is running on `http://127.0.0.1:5000`.
